Overview

FantDev Trading Platform is a comprehensive trading bot management system featuring Telegram integration, web portals, and automated reporting.

Key Features

  • Real-time transaction monitoring
  • Customer balance tracking
  • Automated alerts and notifications
  • Web-based admin and customer portals
  • RESTful API for integrations
  • Web Analysis System: Intelligent market monitoring with competitor analysis and trading signal detection
  • Market Intelligence: Real-time sentiment analysis from cryptocurrency news sources
  • Business Intelligence: Automated competitor tracking and partnership opportunity identification

Quick Start

# Clone the repository
git clone https://github.com/brendadeeznuts1111/myfilterbot.git
cd myfilterbot

# Install Python dependencies
pip install -r requirements.txt

# Install Bun runtime for Web Analysis System (required)
curl -fsSL https://bun.sh/install | bash
bun install

# Start the bot
python3 main_bot.py

# Start the web portal
python3 unified_server.py

# Demo the Web Analysis System
bun demo_web_analysis.ts

Web Analysis System

The Web Analysis System is a powerful intelligence platform that leverages Bun's HTMLRewriter API for efficient web content monitoring, trading signal detection, and competitor analysis. It provides real-time market intelligence to enhance your trading bot's decision-making capabilities.

Core Architecture

HTMLAnalyzer Engine

High-performance content extraction using Bun's HTMLRewriter API

  • Link Analysis: Categorizes links into trading, news, and social media
  • Content Processing: Real-time sentiment analysis and keyword detection
  • Data Extraction: Structured extraction of prices, signals, and indicators
  • Contact Intelligence: Business development opportunity identification

Worker Management System

Multi-threaded processing with Bun's optimized worker threads (500x faster postMessage)

  • Concurrent Analysis: Process multiple websites simultaneously
  • Priority Queuing: High-priority signals processed first
  • Error Handling: Robust retry mechanisms and error recovery
  • Performance Monitoring: Real-time metrics and benchmarking

Key Features

  • Real-time Market Intelligence: Monitor news sources for trading-relevant information
  • Competitor Monitoring: Track competitor websites for feature changes and announcements
  • Signal Detection: Automated trading signal recognition with confidence scoring
  • Sentiment Analysis: Market sentiment classification (positive/negative/neutral)
  • Business Intelligence: Contact extraction for partnership opportunities
  • Portal Integration: Real-time updates to customer and admin portals

Performance Benefits

Configuration & Setup

Complete guide to setting up and configuring the Web Analysis System.

Installation Requirements

Bun Runtime (Required)

The Web Analysis System requires Bun v1.2.21+ for optimal performance:

# Install Bun runtime
curl -fsSL https://bun.sh/install | bash

# Verify installation
bun --version  # Should be 1.2.21 or higher

# Install project dependencies
bun install

Environment Configuration

# Web Analysis Environment Variables
WEB_ANALYSIS_ENABLED=true
WEB_ANALYSIS_WORKERS=4
WEB_ANALYSIS_BATCH_SIZE=20
WEB_ANALYSIS_TIMEOUT=45000

# Monitoring Configuration
MONITOR_NEWS_SOURCES=true
MONITOR_COMPETITORS=true
SIGNAL_DETECTION_ENABLED=true

# Performance Settings
ENABLE_WORKER_THREADS=true
ENABLE_BATCHIN_PROCESSING=true

Integration Examples

Customer Portal Integration

// Enhanced customer portal with live market data
import { WebAnalysisManager } from './src/web_analysis_manager';

const analysisManager = new WebAnalysisManager();

// Get market sentiment for dashboard
const sentiment = await analysisManager.getMarketSentiment();
updateCustomerDashboard({
    marketSentiment: sentiment.overall,
    lastUpdated: sentiment.lastUpdated,
    confidence: sentiment.confidence
});

// Subscribe to high-confidence signals
analysisManager.on('highConfidenceSignal', (signal) => {
    notifyCustomer(customerId, {
        type: 'trading-opportunity',
        message: `${signal.action.toUpperCase()} signal detected for ${signal.symbol}`,
        confidence: (signal.confidence * 100).toFixed(1) + '%'
    });
});

Telegram Bot Integration

// Automated signal notifications via Telegram
import { TradingBotWebIntegration } from './trading_bot_web_integration';

const webIntegration = new TradingBotWebIntegration();

// Automatic customer notifications for high-confidence signals
webIntegration.on('tradingSignal', async (signal) => {
    if (signal.confidence >= 0.7) {
        const message = `🚨 *Trading Signal Alert*\n\n` +
                       `Signal: ${signal.signal}\n` +
                       `Confidence: ${(signal.confidence * 100).toFixed(1)}%\n` +
                       `Source: ${signal.source}\n` +
                       `Time: ${new Date(signal.timestamp).toLocaleString()}`;
        
        await bot.sendMessage(adminChatId, message, {
            parse_mode: 'Markdown',
            reply_markup: {
                inline_keyboard: [[
                    { text: '📊 View Analysis', url: `${portalUrl}/analysis/${signal.id}` },
                    { text: '📢 Notify Customers', callback_data: `notify_signal_${signal.id}` }
                ]]
            }
        });
    }
});
