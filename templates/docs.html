Overview

FantDev Trading Platform is a comprehensive trading bot management system featuring Telegram integration, web portals, and automated reporting.

Key Features

  • Real-time transaction monitoring
  • Customer balance tracking
  • Automated alerts and notifications
  • Web-based admin and customer portals
  • RESTful API for integrations

Quick Start

# Clone the repository
git clone https://github.com/brendadeeznuts1111/myfilterbot.git
cd myfilterbot

# Install dependencies
pip install -r requirements.txt

# Start the bot
python3 main_bot.py

# Start the web portal
python3 unified_server.py

Customer Commands

/start

Initialize the bot and display the welcome message.

/register [customer_id] [password]

Register your account with the bot.

Example: /register BB1042 N9H9

/balance

Check your current balance and P&L.

API Authentication

POST /api/login

Authenticate and receive a JWT token.

{
  "customer_id": "BB1042",
  "password": "N9H9"
}

Customers API

GET /api/customer/{customer_id}

Retrieve customer information and balance.

GET /api/stats

Get global platform statistics.
